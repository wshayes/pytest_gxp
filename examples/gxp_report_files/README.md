# GxP Report Files

This directory contains example output files generated by the pytest-gxp plugin.

## Files

### Traceability Matrix

- **traceability_matrix.csv** - Traceability matrix in CSV format
- **traceability_matrix.md** - Traceability matrix in Markdown format

### Validation Reports

- **csv_validation_report.json** - CSV validation summary report in JSON format
- **csv_validation_report.md** - CSV validation summary report in Markdown format (generated by plugin)

## Usage

These files serve as examples of the output formats generated by the plugin. When you run:

```bash
pytest --gxp --gxp-spec-files=examples/gxp_spec_files --gxp-report-files=examples/gxp_report_files
```

The plugin will generate similar files in the specified report directory.

## PDF Conversion

To convert markdown reports to PDF:

```bash
just docs-pdf
```

This requires the PDF dependencies:
```bash
pip install weasyprint markdown
```
