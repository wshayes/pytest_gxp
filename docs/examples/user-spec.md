# User Specification Example

This is an example user specification file.

```markdown
# User Specification

## Version: 1.0

### US-001: Secure User Access

#### Description
As a user, I need to securely access the system.

1. Log in using username and password
2. Receive feedback on login failures
3. Reset password if forgotten

Expected Result: Users can securely access the system.

#### Metadata
Priority: High
Category: Security
Owner: End Users
```
