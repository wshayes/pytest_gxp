# Functional Specification Example

This is an example functional specification file.

```markdown
# Functional Specification

## Version: 1.0

### FS-001: User Login Functionality

#### Description
The application shall allow authorized users to log in.

1. Display login form
2. Validate credentials
3. Create secure session
4. Redirect to dashboard

Expected Result: User is authenticated and granted access.

#### Metadata
Priority: High
Category: Authentication
Owner: Product Team
```
