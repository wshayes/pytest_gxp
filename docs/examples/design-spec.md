# Design Specification Example

This is an example design specification file.

# Design Specification

## Version: 1.0

### DS-001: User Authentication System

#### Description
The system shall provide a secure user authentication mechanism.

1. Validate user credentials
2. Implement password encryption
3. Support session management
4. Log authentication attempts

#### Metadata
Priority: High
Category: Security
Owner: Security Team
